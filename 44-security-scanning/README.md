# Security Scanning - Vulnerability Assessment

Automated security scanning for container images, configurations, and runtime security in Kubernetes.

## Learning Path

### [01-basic](./01-basic/) - Security Scanning Fundamentals
- Image vulnerability scanning
- Configuration security checks
- Basic security policies
- Admission controller security

### [02-intermediate](./02-intermediate/) - Advanced Security Scanning
- Runtime security monitoring
- Network security scanning
- Compliance scanning
- Custom security policies

### [03-advanced](./03-advanced/) - Enterprise Security Governance
- Continuous security monitoring
- Security automation
- Threat detection and response
- Security metrics and reporting

## Scanning Types

- **Image Scanning** - CVE detection in container images
- **Config Scanning** - Kubernetes manifest security
- **Runtime Scanning** - Live workload monitoring
- **Network Scanning** - Traffic analysis and policies
