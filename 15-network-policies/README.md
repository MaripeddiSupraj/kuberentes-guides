# Network Policies - Network Security and Isolation

Network Policies control traffic flow between Pods, providing network-level security and microsegmentation.

## Learning Path

### [01-basic](./01-basic/) - Network Policy Fundamentals
- Creating Network Policies
- Ingress and egress rules
- Pod and namespace selectors
- Default deny policies

### [02-intermediate](./02-intermediate/) - Advanced Network Security
- Complex selector combinations
- Port-based policies
- External traffic control
- Policy troubleshooting

### [03-advanced](./03-advanced/) - Enterprise Network Security
- Zero-trust networking
- Multi-tenant isolation
- Compliance policies
- Network policy automation

## Policy Types

- **Ingress** - Controls incoming traffic to Pods
- **Egress** - Controls outgoing traffic from Pods
- **Default Deny** - Blocks all traffic by default
- **Allow Specific** - Permits only defined traffic
