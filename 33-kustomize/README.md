# Kustomize - Configuration Management

Kustomize provides template-free customization of Kubernetes YAML configurations.

## Learning Path

### [01-basic](./01-basic/) - Kustomize Fundamentals
- Base and overlay structure
- Resource patching
- ConfigMap and Secret generators
- Common labels and annotations

### [02-intermediate](./02-intermediate/) - Advanced Kustomization
- Strategic merge patches
- JSON patches
- Multi-environment management
- Component composition

### [03-advanced](./03-advanced/) - Enterprise Configuration
- GitOps with Kustomize
- Plugin development
- Validation and testing
- CI/CD integration

## Key Features

- **Declarative** - Pure YAML, no templating
- **Layered** - Base + overlays for environments
- **Generators** - Dynamic ConfigMap/Secret creation
- **Transformers** - Modify resources systematically
