# Service Mesh - Advanced Service Communication

Service mesh provides infrastructure layer for service-to-service communication with security, observability, and traffic management.

## Learning Path

### [01-basic](./01-basic/) - Service Mesh Fundamentals
- What is a service mesh
- Istio installation
- Sidecar proxy pattern
- Basic traffic routing

### [02-intermediate](./02-intermediate/) - Advanced Service Mesh
- Traffic management and routing
- Security policies (mTLS)
- Observability and tracing
- Circuit breaking and retries

### [03-advanced](./03-advanced/) - Enterprise Service Mesh
- Multi-cluster service mesh
- Advanced security policies
- Performance optimization
- Service mesh federation

## Key Components

- **Data Plane** - Sidecar proxies (Envoy)
- **Control Plane** - Configuration and management
- **Ingress Gateway** - External traffic entry
- **Service Registry** - Service discovery
