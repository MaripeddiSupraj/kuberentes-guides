# Backup and Restore - Data Protection

Kubernetes backup and restore strategies for protecting cluster state, persistent data, and application configurations.

## Learning Path

### [01-basic](./01-basic/) - Backup Fundamentals
- etcd backup and restore
- Persistent volume backups
- Configuration backup
- Basic restore procedures

### [02-intermediate](./02-intermediate/) - Advanced Backup Strategies
- Automated backup schedules
- Cross-region backups
- Application-consistent backups
- Backup validation

### [03-advanced](./03-advanced/) - Enterprise Data Protection
- Multi-cluster backup
- Compliance and retention
- Disaster recovery testing
- Backup monitoring

## Backup Types

- **Cluster State** - etcd, configurations
- **Persistent Data** - PV snapshots, databases
- **Applications** - Manifests, secrets, configs
- **Metadata** - RBAC, policies, CRDs
