# CI/CD - Continuous Integration and Deployment

CI/CD pipelines automate building, testing, and deploying applications to Kubernetes clusters.

## Learning Path

### [01-basic](./01-basic/) - CI/CD Fundamentals
- Pipeline concepts
- GitHub Actions for Kubernetes
- Basic build and deploy
- Container registry integration

### [02-intermediate](./02-intermediate/) - Advanced CI/CD
- Multi-environment pipelines
- Testing strategies
- Security scanning
- Rollback mechanisms

### [03-advanced](./03-advanced/) - Enterprise CI/CD
- Multi-cluster deployments
- Compliance and governance
- Advanced deployment strategies
- Pipeline as Code

## Pipeline Stages

- **Build** - Compile and package applications
- **Test** - Unit, integration, security tests
- **Deploy** - Deploy to Kubernetes
- **Monitor** - Observe deployment health
