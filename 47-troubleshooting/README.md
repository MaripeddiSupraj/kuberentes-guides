# Troubleshooting - Problem Diagnosis

Kubernetes troubleshooting techniques and tools for diagnosing and resolving cluster and application issues.

## Learning Path

### [01-basic](./01-basic/) - Basic Troubleshooting
- kubectl debugging commands
- Pod lifecycle issues
- Service connectivity problems
- Resource constraints

### [02-intermediate](./02-intermediate/) - Advanced Diagnostics
- Network troubleshooting
- Storage issues
- Performance problems
- Security-related issues

### [03-advanced](./03-advanced/) - Expert Troubleshooting
- Cluster-level issues
- Control plane problems
- Custom resource debugging
- Production incident response

## Common Issues

- **Pod Issues** - CrashLoopBackOff, ImagePullBackOff
- **Network Issues** - Service discovery, DNS
- **Storage Issues** - PVC binding, mount failures
- **Resource Issues** - CPU/memory limits, quotas
