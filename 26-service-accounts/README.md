# Service Accounts - Pod Identity

Service Accounts provide identity for Pods and control access to the Kubernetes API.

## Learning Path

### [01-basic](./01-basic/) - Service Account Fundamentals
- Creating Service Accounts
- Pod authentication
- Default Service Account
- Token mounting

### [02-intermediate](./02-intermediate/) - Advanced Identity Management
- Custom Service Accounts
- Token lifecycle management
- Cross-namespace access
- Service Account tokens

### [03-advanced](./03-advanced/) - Enterprise Identity
- External identity providers
- Workload identity federation
- Service mesh integration
- Identity governance

## Key Concepts

- **Authentication** - Who is making the request
- **Authorization** - What they can do (RBAC)
- **Tokens** - JWT tokens for API access
- **Mounting** - Automatic token injection
