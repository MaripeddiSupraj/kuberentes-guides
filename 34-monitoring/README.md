# Monitoring - Metrics and Observability

Kubernetes monitoring provides visibility into cluster health, resource usage, and application performance.

## Learning Path

### [01-basic](./01-basic/) - Monitoring Fundamentals
- Metrics Server installation
- Resource metrics (CPU, memory)
- kubectl top commands
- Basic monitoring setup

### [02-intermediate](./02-intermediate/) - Advanced Monitoring
- Prometheus and Grafana
- Custom metrics
- ServiceMonitor resources
- Alert rules

### [03-advanced](./03-advanced/) - Enterprise Monitoring
- Multi-cluster monitoring
- Long-term storage
- SLI/SLO monitoring
- Compliance and governance

## Monitoring Stack

- **Metrics Server** - Resource metrics API
- **Prometheus** - Metrics collection and storage
- **Grafana** - Visualization and dashboards
- **AlertManager** - Alert routing and management
