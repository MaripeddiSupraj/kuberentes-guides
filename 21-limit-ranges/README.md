# Limit Ranges - Default Resource Constraints

Limit Ranges enforce minimum, maximum, and default resource constraints for Pods and containers in a namespace.

- Policy-based limits

## Constraint Types

- **Default** - Applied when not specified
- **DefaultRequest** - Default resource requests
- **Min** - Minimum allowed resources
- **Max** - Maximum allowed resources
