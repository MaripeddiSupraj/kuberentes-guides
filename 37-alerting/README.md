# Alerting - Alert Management

Kubernetes alerting provides proactive notification of issues and anomalies in cluster and application health.

## Learning Path

### [01-basic](./01-basic/) - Alerting Fundamentals
- Alert concepts and terminology
- Prometheus AlertManager
- Basic alert rules
- Notification channels

### [02-intermediate](./02-intermediate/) - Advanced Alerting
- Complex alert expressions
- Alert grouping and routing
- Silence and inhibition rules
- Multi-channel notifications

### [03-advanced](./03-advanced/) - Enterprise Alerting
- SLI/SLO-based alerting
- Alert fatigue management
- Incident response integration
- Compliance and audit alerts

## Alert Components

- **Alert Rules** - Define when to fire alerts
- **AlertManager** - Routes and manages alerts
- **Receivers** - Notification destinations
- **Inhibition** - Suppress dependent alerts
