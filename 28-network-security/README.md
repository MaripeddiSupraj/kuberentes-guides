# Network Security - Network-level Security

Advanced network security configurations and policies for securing Kubernetes cluster communications.

## Learning Path

### [01-basic](./01-basic/) - Network Security Fundamentals
- Network policies basics
- Service mesh security
- TLS/mTLS configuration
- Ingress security

### [02-intermediate](./02-intermediate/) - Advanced Network Security
- Zero-trust networking
- Network segmentation
- Traffic encryption
- Security monitoring

### [03-advanced](./03-advanced/) - Enterprise Network Security
- Multi-cluster security
- Compliance frameworks
- Threat detection
- Security automation

## Security Layers

- **Network Policies** - Pod-to-pod communication control
- **Service Mesh** - mTLS, traffic policies
- **Ingress** - TLS termination, WAF
- **CNI** - Network plugin security features
