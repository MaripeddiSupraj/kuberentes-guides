# Logging - Log Aggregation and Analysis

Kubernetes logging provides centralized log collection, storage, and analysis for cluster and application logs.

## Learning Path

### [01-basic](./01-basic/) - Logging Fundamentals
- Container logging basics
- kubectl logs commands
- Log rotation and retention
- Centralized logging concepts

### [02-intermediate](./02-intermediate/) - Log Aggregation
- Fluentd/Fluent Bit deployment
- Elasticsearch and Kibana
- Log parsing and filtering
- Multi-tenant logging

### [03-advanced](./03-advanced/) - Enterprise Logging
- Log analytics and alerting
- Compliance and audit logs
- Performance optimization
- Long-term retention

## Logging Stack Components

- **Log Agents** - Fluentd, Fluent Bit, Filebeat
- **Storage** - Elasticsearch, Loki, CloudWatch
- **Visualization** - Kibana, Grafana
- **Processing** - Logstash, Vector
