# Deployments - Application Management

Deployments provide declarative updates for Pods and ReplicaSets, enabling rolling updates, rollbacks, and scaling.

## Learning Path

### [01-basic](./01-basic/) - Deployment Fundamentals
- Creating deployments
- Scaling applications
- Basic rolling updates
- Deployment status and history

### [02-intermediate](./02-intermediate/) - Advanced Deployment Strategies
- Rolling update strategies
- Rollback operations
- Deployment annotations and labels
- Resource management in deployments

### [03-advanced](./03-advanced/) - Enterprise Deployment Patterns
- Blue-green deployments
- Canary deployments
- Advanced update strategies
- Deployment hooks and lifecycle

## Key Benefits

- **Declarative updates** - Describe desired state
- **Rolling updates** - Zero-downtime deployments
- **Rollback capability** - Easy reversion to previous versions
- **Scaling** - Horizontal scaling of applications
- **Self-healing** - Automatic Pod replacement
