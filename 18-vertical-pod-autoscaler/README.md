# Vertical Pod Autoscaler - Vertical Scaling

VPA automatically adjusts CPU and memory requests for containers based on usage.

## Update Modes

- **Off** - Only recommendations, no updates
- **Initial** - Set resources on Pod creation
- **Auto** - Update running Pods automatically
