# Pod Security Standards - Modern Pod Security

Pod Security Standards define security policies for Pods using built-in admission controller.

## Learning Path

### [01-basic](./01-basic/) - Pod Security Fundamentals
- Security levels (Privileged, Baseline, Restricted)
- Namespace-level enforcement
- Pod Security admission controller
- Migration from PSPs

### [02-intermediate](./02-intermediate/) - Advanced Security Policies
- Custom security contexts
- Security policy violations
- Exemptions and overrides
- Multi-tenant security

### [03-advanced](./03-advanced/) - Enterprise Security Governance
- Policy automation
- Compliance frameworks
- Security scanning integration
- Zero-trust pod security

## Security Levels

- **Privileged** - Unrestricted policy
- **Baseline** - Minimally restrictive
- **Restricted** - Heavily restricted, hardened
