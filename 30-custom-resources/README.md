# Custom Resources - Extending Kubernetes API

Custom Resource Definitions (CRDs) extend the Kubernetes API with custom resource types.

## Learning Path

### [01-basic](./01-basic/) - CRD Fundamentals
- Creating Custom Resource Definitions
- Custom Resource instances
- Schema validation
- API versioning

### [02-intermediate](./02-intermediate/) - Advanced CRD Features
- OpenAPI schema validation
- Subresources (status, scale)
- Multiple versions
- Conversion webhooks

### [03-advanced](./03-advanced/) - Enterprise Custom Resources
- Custom controllers
- Operator patterns
- API aggregation
- Custom resource governance

## Key Components

- **CRD** - Defines the custom resource schema
- **CR** - Instance of the custom resource
- **Controller** - Watches and acts on custom resources
- **Validation** - Schema-based validation rules
