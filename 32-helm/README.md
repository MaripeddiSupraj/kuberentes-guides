# Helm - Package Management

Helm is the package manager for Kubernetes, providing templating, versioning, and release management.

## Learning Path

### [01-basic](./01-basic/) - Helm Fundamentals
- Installing Helm
- Creating charts
- Templates and values
- Installing releases

### [02-intermediate](./02-intermediate/) - Advanced Helm Usage
- Chart dependencies
- Hooks and tests
- Custom functions
- Repository management

### [03-advanced](./03-advanced/) - Enterprise Helm Patterns
- Multi-environment deployments
- GitOps with Helm
- Security and signing
- Custom operators with Helm

## Key Concepts

- **Charts** - Helm packages
- **Templates** - Kubernetes YAML with variables
- **Values** - Configuration parameters
- **Releases** - Deployed chart instances
