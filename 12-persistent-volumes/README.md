# Persistent Volumes - Advanced Storage Management

Persistent Volumes (PV) and Persistent Volume Claims (PVC) provide storage abstraction and lifecycle management.

## Learning Path

### [01-basic](./01-basic/) - PV/PVC Fundamentals
- Creating Persistent Volumes
- Persistent Volume Claims
- Storage classes
- Volume binding modes

### [02-intermediate](./02-intermediate/) - Advanced Storage Patterns
- Dynamic provisioning
- Storage classes with parameters
- Volume expansion
- Snapshot and restore

### [03-advanced](./03-advanced/) - Enterprise Storage
- Multi-zone storage
- Storage monitoring
- Backup strategies
- Performance optimization

## Storage Types

- **hostPath** - Node local storage
- **nfs** - Network File System
- **csi** - Container Storage Interface
- **cloud** - AWS EBS, GCP PD, Azure Disk
