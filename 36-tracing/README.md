# Tracing - Distributed Tracing

Distributed tracing tracks requests across multiple services to understand system behavior and performance.

## Learning Path

### [01-basic](./01-basic/) - Tracing Fundamentals
- Distributed tracing concepts
- Jaeger installation
- Basic trace collection
- Trace visualization

### [02-intermediate](./02-intermediate/) - Advanced Tracing
- Custom instrumentation
- Sampling strategies
- Performance analysis
- Error tracking

### [03-advanced](./03-advanced/) - Enterprise Tracing
- Multi-cluster tracing
- Long-term storage
- Alerting on traces
- Compliance and privacy

## Tracing Components

- **Tracer** - Generates spans and traces
- **Collector** - Receives and processes traces
- **Storage** - Stores trace data
- **UI** - Visualizes traces and analytics
